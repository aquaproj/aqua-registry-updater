# aqua-registry-updater

Renovate alternative only for aqua-registry. Overcome Renovate's scalability issue

## GitHub Access Token

- `GITHUB_TOKEN`
  - `pull-requests:write`
  - `contents:write`
- `AQUA_REGISTRY_UPDATER_CONTAINER_REGISTRY_TOKEN`
  - classic personal access token
  - `write:packages`
  - `read:org`

## Requirements

- GitHub CLI
- int128/ghcp
